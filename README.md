## 🏦 | Astralend

### Overview
Astralend is a production-grade, transparent machine learning pipeline that predicts whether a loan application will be **approved** or **rejected**, while providing interpretable explanations that bankers, risk officers and regulators can actually understand. In addition, the system combines solid classical ML models with modern gradient boosting, careful feature engineering, missing value treatment, ordinal & target encoding, monotonic constraints (where appropriate) and SHAP-based explainability.

You can also check out the **[Astralend's website](https://astralend.lovable.app/)** in order to experience it live..
